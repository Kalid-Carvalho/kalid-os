## @kalidcarvalho | Builder — Kalid Carvalho | IA e Tech
*Usar para: conteúdo @kalidcarvalho, carrosséis tech, reels de IA, YouTube, LinkedIn*

### Cores
- **Fundo principal:** #0D1526 (azul escuro)
- **Fundo alternativo:** #F0F4FF (azul gelo, para contraste)
- **Cor de destaque / CTA:** #1A56DB (azul royal)
- **Texto principal:** #FFFFFF (sobre fundo escuro) / #0A0A0A (sobre fundo claro)
- **Texto secundário / handles:** #9BB3E8 (azul muted)
- **Separadores:** #1E2E4A

### Tipografia
- **Fonte:** Space Grotesk (Google Fonts)
- **Hero:** 60–68px, weight 700
- **Heading:** 44px, weight 600
- **Body:** 34px, weight 500
- **Caption:** 24px, weight 400–500
- **Tag/Badge:** 20–22px, weight 600 (uppercase)

### Elementos-chave
- Border-radius: 8px em badges e botões
- Border accent: 4–6px left border em #1A56DB nos blocos de destaque
- Separadores: 1px #1E2E4A
- Logo: Monograma KC — quadrado arredondado #1A56DB, "K" bold + "C" light 80% opacidade

### O que NUNCA fazer
- Fundo branco ou cinza claro como padrão
- Fonte sem serifa diferente de Space Grotesk
- Accent amarelo ou laranja
- Mais de 2 emojis por post
- Emojis em posts educativos/técnicos

## @eukalidcarvalho — Marketing Jurídico para Advogados
*Usar para: conteúdo @eukalidcarvalho, carrosséis jurídicos, landing pages de captação, propostas para advogados*

### Conceito
Institucional, autoritativo, elegante. O dourado comunica peso jurídico, tradição e seriedade — sem parecer agência genérica. Contraste entre serif editorial e sans moderno cria a tensão "mundo jurídico + sistema digital".

### Cores
- **Fundo principal:** #0C1018 (preto quente)
- **Fundo secundário:** #081018
- **Surface / cards:** #101928
- **Cor de destaque / CTA:** #C9A84C (dourado)
- **Accent hover:** #D9BC72
- **Texto principal:** #F4F0E8 (cream quente)
- **Texto secundário:** #9A9080
- **Texto muted:** #4A4030
- **Separadores:** #1C1A14
- **Dim overlay accent:** rgba(201,168,76,0.09)
- **Border accent:** rgba(201,168,76,0.28)

### Tipografia
- **Display / headlines:** Cormorant Garamond (Google Fonts) — itálico para ênfase, 600–700
- **Body / labels / CTAs:** DM Sans (Google Fonts) — 400/500/600
- **Hero:** clamp(42px, 8vw, 76px), weight 700, line-height 0.95
- **Heading seção:** clamp(28px, 5vw, 48px), weight 700
- **Body:** 15–17px, weight 400–500
- **Labels:** 11px, weight 600, uppercase, letter-spacing 0.14em

### Elementos-chave
- Border-radius: 6–10px em cards, 100px em badges pill
- Botão CTA: fundo #C9A84C, texto #0C1018 (escuro — contraste obrigatório)
- Left border accent: 3px solid #C9A84C em pull quotes
- Grid de pontos no hero: radial-gradient rgba(201,168,76,0.14)
- Glow radial no hero: rgba(201,168,76,0.09)
- Scrollbar: 2px, cor #D9BC72
- Noise texture: opacity 0.3–0.5 (atmosfera)
- Container: max-width 1300px

### Header padrão dos slides
- Avatar circular de Kalid (60–64px, borda 2px #C9A84C)
- Nome: DM Sans 700, 22–24px, #F4F0E8
- Handle: @eukalidcarvalho, 18–20px, #9A9080
- Watermark: @eukalidcarvalho, canto inferior direito, #4A4030

### O que NUNCA fazer
- Misturar com o azul #1A56DB do @kalidcarvalho
- Usar fundo claro — sempre escuro
- Botão dourado com texto branco (contraste insuficiente)
- Tipografia serifada no body (só no display)
- Emojis em posts educativos/jurídicos
- Prometer resultado absoluto (questão OAB)

## GROVW (grovw.com.br)
*Usar para: site, propostas, materiais de vendas, ads GROVW*

### Cores
- **Fundo principal:** #000000 (preto)
- **Fundo seções:** #0f0f0f
- **Surface / cards:** #32373c
- **Cor de destaque / CTA:** #cdff00 (limão neon)
- **Accent hover:** #d0ff0f
- **Texto principal:** #ffffff
- **Texto secundário:** #888888
- **Bordas:** rgba(255,255,255,0.1)

### Tipografia
- **Fonte principal:** Inter (sans-serif)
- **Fonte mono / técnica:** JetBrains Mono
- **Estilo:** uppercase tracking-widest para labels e nav, font-bold tracking-tighter para headings

### Estilo geral
Digital Architect — dark, tech, high-performance. Grid overlay, noise texture, kinetic 3D cards, marquee, parallax. Sensação de produto de engenharia premium.

### Elementos-chave
- Border-radius: 4–8px (conservador)
- Botões CTA: bg #cdff00, texto preto, uppercase, beam hover effect
- Cards: flashlight hover (radial gradient ao mouse), border #cdff00 no hover
- Separadores: gradient linear com #cdff00 a 50% opacidade
- Animações: reveal (opacity + translateY), marquee, spin-slow, draw path SVG

### O que NUNCA fazer (GROVW)
- Fundo claro ou branco
- Cores vibrantes além do accent limão
- Tipografia serifada
- Layout sem grid estrutural
- Botões arredondados demais (pill shape)

## Observações adicionais
- Sempre identificar qual marca é o output antes de escolher o design system
- Em caso de dúvida, perguntar: "Isso é pra GROVW ou pra marca pessoal?"
- As duas marcas têm estilos opostos — não misturar elementos entre elas
