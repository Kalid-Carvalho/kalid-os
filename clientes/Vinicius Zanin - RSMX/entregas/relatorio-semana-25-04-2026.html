Integração RVOps · detalhe técnico

O formulário do site já está capturando leads e enviando notificação automática via WhatsApp. O próximo passo é fazer esses leads entrarem automaticamente no RVOps para gestão e follow-up centralizado.

Para configurar isso, precisamos de uma das seguintes opções do lado da plataforma:

Opção A: Webhook do RVOps: uma URL onde enviamos os dados do lead via POST (nome, telefone, email, origem da campanha).
Opção B: API Key do RVOps: se a plataforma tiver API REST, conectamos diretamente.

Com qualquer uma das duas, configuramos o N8N para enviar cada lead das campanhas para o RVOps em menos de 5 segundos após o preenchimento.
